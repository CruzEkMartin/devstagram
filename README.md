<h1 align="center">Devstagram</h1>


<br/>

## Acerca de DevJobs
<br/>
1er Proyecto del Curso de Laravel 9 de Juan Pablo de la Torre. Clon de Instagram para developers realizado con PHP 8 , MVC, Tailwind Css, Livewire y Dropzone.
